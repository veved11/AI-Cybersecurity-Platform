# AI-Cybersecurity-Platform

AI-powered cybersecurity monitoring platform built with FastAPI, PostgreSQL, Machine Learning, and Bootstrap.

## Features

- User Authentication
- User Management
- Security Alerts Management
- Log File Upload
- AI Threat Detection
- Risk Scoring System
- Threat Statistics Dashboard
- PDF Security Reports
- Machine Learning Classification

## Tech Stack

- FastAPI
- Python
- PostgreSQL
- SQLAlchemy
- Scikit-Learn
- Bootstrap
- Chart.js
- ReportLab
